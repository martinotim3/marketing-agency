# Foxtail Inc — Homepage Mockups (4 Options)

## Purpose
Pick a visual direction for Foxtail Inc marketing agency site before building full 5-page site + VPS deploy.

## Scope
4 standalone static HTML homepage mockups, same brand/content skeleton, different visual treatment. No shared build tooling — plain HTML/CSS/JS, one self-contained file per option, viewable directly in browser.

## Brand inputs
- Name: Foxtail Inc
- Vibe: playful/creative
- Palette: foxtail-inspired — rust/orange + cream
- Services: social media, SEO, ads, branding, web design (full-service)
- Sections per mockup: hero, services, portfolio teaser, about blurb, contact CTA, footer

## The 4 directions
1. **Bold Editorial** — huge serif headlines, magazine grid, rust/cream color blocks
2. **Organic Motion** — blob shapes, hand-drawn accents, animated hover states, rounded everything
3. **Grid Punk** — sharp geometric grid, mono+display font mix, orange-on-cream, brutalist-lite
4. **Retro Warm** — vintage poster feel, textured cream bg, rust typography, sun/fox-tail motif

## Output
`mockups/01-bold-editorial.html`, `02-organic-motion.html`, `03-grid-punk.html`, `04-retro-warm.html` — each self-contained (inline CSS, no external deps except Google Fonts).

## Next steps (not this phase)
- User picks winning direction
- Build full 5-page static site (Home/Services/Portfolio/About/Contact) in that style
- VPS hosting: nginx setup, domain/SSL, deploy — details TBD when reached
